# Produce the public Ashen Voice installer

Players do **not** install CMake, Visual Studio, Node.js, or any command-line tools.
Those tools are used only once by the automated Windows build system.

## Build through GitHub

1. Create a private or public GitHub repository.
2. Upload this project, including the hidden `.github` folder.
3. Open the repository's **Actions** tab.
4. Select **Build Ashen Voice Windows Installer**.
5. Choose **Run workflow**.
6. Download the **Ashen-Voice-Windows-Installer** artifact.
7. Distribute the enclosed `AshenVoice-Setup-1.1.0.exe` file.

A release can also be generated by pushing a version tag such as `v1.1.0`.
The workflow then attaches the installer and SHA-256 checksum to a GitHub Release.

## Player experience

1. Download `AshenVoice-Setup-1.1.0.exe`.
2. Run the installer.
3. Open Ashen Voice.
4. Enter the Discord bot token, server ID, and voice-channel ID.
5. Click **Start Overlay**.
6. Launch OctoWoW.

No build tools are installed on the player's computer.
