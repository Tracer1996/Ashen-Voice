ASHEN VOICE 1.4.4 - FULL REPOSITORY

1. Extract Ashen-Voice-1.4.4-Full-Repository.zip.
2. Open GitHub Desktop and select Ashen-Voice.
3. Click Repository > Show in Explorer.
4. Delete everything in the repository folder except the hidden .git folder.
5. Copy every file and folder from the extracted AshenVoice folder into the repository folder.
6. In GitHub Desktop, commit: Replace repository with complete Ashen Voice 1.4.4 project
7. Push origin.
8. Open GitHub > Actions.
9. Run Build Ashen Voice 1.4.4.
10. Download the Ashen-Voice-1.4.4-Installer artifact.
11. Extract and run AshenVoice-Setup-1.4.4.exe.
