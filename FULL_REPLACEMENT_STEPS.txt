ASHEN VOICE 1.4.3 - FULL REPOSITORY

This ZIP contains the entire project, not a patch.

1. Extract the ZIP.
2. Open GitHub Desktop.
3. Select Ashen-Voice.
4. Click Repository > Show in Explorer.
5. Delete everything in the repository folder EXCEPT the hidden .git folder.
6. Copy every file and folder from the extracted AshenVoice folder into the repository folder.
7. In GitHub Desktop, commit: Replace repository with complete Ashen Voice 1.4.3 project
8. Push origin.
9. Open GitHub > Actions.
10. Run Build Ashen Voice 1.4.3.
11. Download the Ashen-Voice-1.4.3-Installer artifact.
12. Extract and run AshenVoice-Setup-1.4.3.exe.
