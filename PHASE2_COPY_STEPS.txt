ASHEN VOICE PHASE 2 - COPY STEPS

1. Extract this ZIP.
2. Open GitHub Desktop.
3. Select the Ashen-Voice repository.
4. Click Repository > Show in Explorer.
5. Keep that repository folder open.
6. Open the extracted Ashen-Voice folder from this ZIP.
7. Press Ctrl+A, then Ctrl+C.
8. Go back to the repository folder and press Ctrl+V.
9. Choose Replace the files in the destination when Windows asks.
10. Open the repository folder's .github\workflows folder.
11. Delete every old .yml file except build-phase2.yml.
12. Return to GitHub Desktop.
13. Enter this summary: Add Phase 2 DirectX 9 test overlay
14. Click Commit to main.
15. Click Push origin.
16. Open the repository on GitHub and click Actions.
17. Open Build Ashen Voice Phase 2.
18. Click Run workflow, then Run workflow again.
19. When the build is green, open it and download Ashen-Voice-Phase2-Installer.
20. Extract the downloaded artifact and run AshenVoice-Setup-1.1.0.exe.
