ASHEN VOICE PHASE 3.1 COPY STEPS

1. Extract this ZIP.
2. Open GitHub Desktop and select Ashen-Voice.
3. Click Repository > Show in Explorer.
4. Copy all files from this extracted folder into the existing repository folder.
5. Choose Replace files in the destination.
6. Do not delete the hidden .git folder.
7. In .github/workflows, keep build-phase3-1-local-rpc.yml and remove the older Phase 3 workflow after confirming the new file copied correctly.
8. Complete DISCORD_SETUP.md.
9. Commit: Add Phase 3.1 local Discord connection
10. Push origin.
11. Run the GitHub Actions workflow Build Ashen Voice Phase 3.1 Local Discord.
12. Download the Ashen-Voice-Phase3.1-Local-Discord-Installer artifact.
13. Extract it and run AshenVoice-Setup-1.3.0.exe.
