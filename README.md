# Ashen Voice 1.4.2

Discord speaker overlay for 32-bit DirectX 9 World of Warcraft clients.

## Features

- Fullscreen DirectX 9 overlay
- Direct connection to the local Discord desktop app
- Discord avatars and display names
- Live overlay settings
- Four screen positions
- Scale, width, avatar, text, opacity, ring, speaker limit, and hide-delay controls
- Higher-resolution avatar and text rendering
- Automatic administrator request on launch
- Elevated Start with Windows task
- System tray support

## Build

1. Add the repository variable `DISCORD_CLIENT_ID` in GitHub.
2. Open **Actions**.
3. Run **Build Ashen Voice Phase 4.2**.
4. Download **Ashen-Voice-Phase4-Installer**.
5. Extract and run `AshenVoice-Setup-1.4.2.exe`.

## Use

1. Open Discord and join a voice channel.
2. Open Ashen Voice and approve the administrator prompt.
3. Click **Connect Discord**.
4. Launch WoW.
5. Click **Start Overlay**.
6. Open **Overlay Settings** to adjust the overlay.
