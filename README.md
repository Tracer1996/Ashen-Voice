# Ashen Voice 1.4.5

Fullscreen Discord speaker overlay for classic World of Warcraft clients.

## Included

- Direct Discord desktop connection
- DirectX 9 fullscreen overlay
- Discord avatars and display names
- Discord bot filtering
- Persistent overlay preview
- Position, offset, size, opacity, and speaker settings
- Automatic administrator request
- Single-instance protection
- Safer speaker-state file updates
